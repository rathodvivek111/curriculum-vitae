text digitization

Description

Our aim is to analyze the process and algorithms used in detecting the text written in captured image. There are some
limitations of this android app like image quality should be good enough to detect the text given in the image. First, this app captures the image and then using machine learning algorithm it scan the whole image and figured out each character written in the image as text. For Android app of text digitization, we have created user friendly interface. Simply user has to click the image containing text. After clicking image when user click the button, image is converted into bitmap and one file is created into which all the characters are copied and this file will be in editable form. By implementing this project, we want to analyze the accuracy and efficiency of the particular API (Google Vision API) which we have used in this project. We also have compared the output of this API with outputs of other available APIs like Microsoft API, ICR API etc. Our goal is to learn machine learning models and different algorithms used in this API to detect the text from given image. It can be useful as a research topic to develop our own algorithms to detect text. We can learn the procedures and different machine learning algorithms used in these implementations to make our own API also.
