Description

Snake Xenzia is a well-known arcade video game where a player manoeuvre a line of blocks that represents snake which lengthens on capturing a small square block that represents its food. The goal of this game is to capture highest number of food items while avoiding collision with the outer walls of the area as well as internal snake collide i.e. self-collision. The core idea behind the game lies in the fact that every image/visual thing consist of tiny pixels and these pixels when combined in large number forms a block that is
visible to human eye. These blocks can further be placed parallel to form a line of blocks and represent a linear object like a snake. A similar block can be used to represent snakeâ€™s food. When this linear pattern overlaps the single block, the snake is said to have swallowed food. Whereas a normal snake game is played manually by user, the objective of this project is to develop AI agent that will play the game i.e. the computer plays on its own collecting maximum number of fruits while avoiding collision. The project is based on concepts of Reinforcement Learning, which is one of types of machine learning used for creating Artificial Intelligence systems.
