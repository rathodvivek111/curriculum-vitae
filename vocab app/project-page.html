Vocabulary Learning App

Description

This is the app to improve the English language. This app consists of a database of user name with their Email id. Another database, which consists of questions about English language by which user can improve his/her language. Third database is about word history. After signing in, when user start the level, the history is created of words which is used by the user. There will be different levels about vocabulary questions in which user has to give the correct answer. Without clearing the current he/she cannot go ahead for the next level. There will be classification of questions according to its difficulty as easy, medium and hard. For good practice, user can start with easy level and go upto hard level. After completing all the levels he/she will be able to improve his/her language in speaking and writing.
