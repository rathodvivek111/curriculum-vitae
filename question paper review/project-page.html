Online question papers feedback

Description

We have to create a web site to get the feedback of the semester end question paper from the student. The data of the website is stored in SQL server. The whole access of the website is given to the Admin. Admin can create access for faculty and student. Faculty have access to add or remove question paper and questions related to the paper. Student have access to fill up the feedback which is given from faculty. We created many pages in asp.net using aspx language and designed it and give it animation using w3.css language. We also used my sql database for storing data. First of all, we made a page for login for the student and faculty to use the site. We also make registration forms for new entries of student and faculty. We also make one page for the user if he or she forgotten password than recover it by using this facility. We mainly use data binding for the convenience of the user so user can get easily what he or she wants. All other things are explained in detail with images in below chapters. Main objective of the website is to get the review of semester end question paper. The review is used to stabilize the difficulty of the semester end question paper.
